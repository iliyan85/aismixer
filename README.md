# AISMixer Website Branch

This branch contains the public AISMixer website for GitHub Pages.

GitHub Pages deploys from the `website` branch using `docs/` as the site root.
Runtime Python code, runtime configuration, deployment scripts, and tests belong
on the `main` branch.

Kept here:

- `docs/` - public website content and GitHub Pages/Jekyll files
- `SITE_WORKFLOW.md` - site branch workflow notes
- `.gitattributes` - repository line-ending rules
- `.gitignore` - local ignore rules
- `LICENSE` - project license
